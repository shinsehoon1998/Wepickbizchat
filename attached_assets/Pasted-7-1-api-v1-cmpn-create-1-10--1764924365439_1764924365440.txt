7.1 캠페인 생성 (/api/v1/cmpn/create)
캠페인을 [승인 요청] 상태로 생성.
중요 규칙

시작 시간 규칙: 현재 시간 대비 1시간 이후, 10분 단위로 체크

10:11에 11:15 시작 캠페인 → 실패
10:11에 11:20 시작 캠페인 → 성공


billingType별 메시지 생성 규칙

billingType값규칙LMS0mms 데이터만 필요, 파일 없음RCS MMS1mms + rcs 모두 필요, mms에 파일 있음 (RCS 실패시 MMS fallback)MMS2mms 데이터만 필요, 파일 있음RCS LMS3mms + rcs 모두 필요, mms에 파일 없음 (RCS 실패시 LMS fallback)

URL분석 길이 변환

개발기: [URL분석] 하나당 32자 증가
상용기: [URL분석] 하나당 25자 증가


코드삽입 길이 변환

[코드삽입] 하나당: <쿠폰문자열길이> - 6 만큼 증가
